Термохімія і питома теплоємність
Екпериментальне визначення питомої теплоємності розчину 2

1. Ви працюєте хіміком-аналітиком в компанії, яка намагається розробити охолоджуючі рідини для двигунів, що є більш ефективними за традиційні аналоги на основі етиленгліколю. Колега приніс Вам розчин із назвою "Розчин сполуки A" для перевірки. Але цього разу густина даного розчину невідома.

    a) Визначте питому теплоємність розчину сполуки A, використовуючи Virtual Lab.

    Примітка: Терези Ви можете знайти, натиснувши на іконку обладнання, яка розташована по лівому краю робочого столу

(Вважайте, що питома теплоємність води становить 4.184 Дж/г*°C)

У цій роботі, ми припускаємо, що тепло не розсіюється у навколишнє середовище, тому Вам необхідно "ізолювати" робочий посуд. Для ізоляції окремого посуду у Virtual Lab, користувачі Mac повинні отримувати command key при натисканні на посуд; користувачі Windows повинні натиснути правою клавішею миші на посуд. У меню, що з'явиться, треба обрати "Термічні властивості." Поставте галочку у полі "Ізолювати від навколишнього середовища." Температура вмісту такого посуду буде залишатися постійною. Ви також можете використати вікно "Термічні властивості" для встановлення необхідної температури розчинів.
